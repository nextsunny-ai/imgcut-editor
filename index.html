HTML 이미지 편집기

🌐 내 도메인에 배포하기

✅ 1단계: 편집 완료 후 다운로드
위의 ⬇ 다운로드 버튼을 클릭하면
편집된 HTML 파일이 PC에 저장됩니다.
✅ 2단계: 호스팅 업체에 업로드
도메인을 구매한 호스팅 업체(카페24, 가비아, 후이즈 등)의
FTP 또는 파일 관리자에 접속하여
다운로드한 HTML 파일을 index.html로 이름을 바꿔서 업로드합니다.
✅ 3단계: 도메인 연결 확인
업로드 후 내 도메인(예: www.example.com)에 접속하여
편집된 페이지가 정상적으로 보이는지 확인합니다.
💡 무료 호스팅을 쓰고 싶다면?
GitHub Pages (무료, 영구)
1. github.com 가입 → 새 저장소(Repository) 생성
2. 다운로드한 파일을 index.html로 업로드
3. Settings → Pages → Branch: main → Save
4. 몇 분 후 username.github.io/저장소명 으로 접속 가능

커스텀 도메인 연결:
Pages 설정에서 Custom domain에 내 도메인 입력 →
도메인 업체(후이즈 등)에서 CNAME 레코드 추가
→ username.github.io 로 연결
📧 도움이 필요하면
배포 과정에서 어려움이 있으시면
편집된 HTML 파일을 첨부하여 관리자에게 전달해 주세요.
대신 배포해 드립니다.
