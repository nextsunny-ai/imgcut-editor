USER GUIDE

써니에디터 사용 가이드

완성된 HTML 페이지의 이미지와 텍스트를 코드 한 줄 건드리지 않고 한 번에 편집합니다. 설치 없음, 회원가입 없음, 서버 없음.

에디터 열기 샘플 1 (단일 페이지) 샘플 2 (3-페이지)

3가지 모드 — 어떤 걸 쓰나요?

HTML

HTML 파일 1개. 모든 CSS·이미지가 그 파일 안에 인라인되어 있는 경우. 가장 단순.

ZIP

사이트 폴더를 ZIP으로 묶어서 업로드. 외부 CSS·이미지·여러 HTML 페이지 모두 지원.

폴더

폴더를 통째로 선택 (Chrome / Edge). ZIP과 동일하지만 압축 단계 생략.

TIP

어떤 모드인지 모르겠다면 ZIP으로 가세요. 가장 안전하고 어떤 사이트든 처리합니다.

사용법 4단계

  1. 모드 선택 — 좌측 사이드바 상단의 [HTML / ZIP / 폴더] 탭에서 모드 선택.
  2. 파일/폴더 업로드 — 드래그 앤 드롭 또는 클릭. ZIP/폴더는 자동으로 메인 HTML을 찾아 열어줍니다.
  3. 편집 — 상단 탭에서 [이미지] / [텍스트] / [미리보기] 전환. 카드 단위로 교체·수정·삭제.
  4. 다운로드 — HTML 모드는 단일 HTML, ZIP/폴더 모드는 ZIP으로 다시 묶여 저장됩니다.

다중 HTML 페이지가 있을 때

ZIP/폴더 안에 index.html, about.html, contact.html 처럼 여러 페이지가 있으면 사이드바에 페이지 선택 드롭다운이 나타납니다. 페이지를 바꿔서 각각 편집한 뒤, 다운로드하면 모든 페이지가 묶인 ZIP이 출력됩니다.

주의: 페이지를 바꾸기 전에 변경 사항이 사라진다는 경고가 나오면, 일단 다운로드한 뒤 다음 페이지로 넘어가세요.

가능한 것 / 불가능한 것

가능

  • HTML 안의 모든 <img> 자동 감지·교체·크롭·삭제·크기 조정
  • 모든 제목·문단·버튼·리스트 텍스트 인라인 편집
  • 외부 CSS 파일 (<link rel="stylesheet">) 자동 인라인 후 다시 분리하여 출력
  • 상대경로 이미지 (./images/logo.png 등) 실제 표시·교체
  • 여러 HTML 페이지 동시 보관 + 페이지 전환 편집
  • 팝업 배너 / 쿠키 동의 1클릭 삽입
  • 변경/삭제/교체 모두 개별 취소 가능
  • 한국어 / 영어 UI 자동 전환

불가능

  • React/Vue/Svelte 컴포넌트 소스코드 자체 (.jsx, .vue). 다만 빌드 후 결과물(정적 HTML)은 처리 가능.
  • 서버에서 동적으로 생성되는 페이지의 템플릿 자체. 받은 정적 HTML만 편집됨.
  • JS로 런타임에 생성되는 콘텐츠는 정적 HTML에 없으면 감지 안 됨.

자주 묻는 질문

업로드한 파일이 서버로 전송되나요?
아니요. 모든 처리는 브라우저 안에서만 일어납니다. 파일·이미지·텍스트는 외부로 전송되지 않습니다.
인터넷 없이 쓸 수 있나요?
네. index.html 파일을 다운로드해서 더블클릭하면 오프라인에서도 작동합니다. 첫 폰트 로딩만 인터넷이 필요합니다.
편집한 결과가 원본을 망가뜨리나요?
원본 파일은 그대로 보존됩니다. 다운로드된 결과물은 이름_edited.html 또는 이름_edited.zip으로 별도 저장됩니다.
모바일에서도 되나요?
기본 기능은 됩니다. 다만 작업 효율을 위해 데스크탑 브라우저(Chrome, Edge, Safari) 권장.
가격은요?
무료입니다. 향후 사이트에 광고가 1개 표시될 수 있습니다(편집 영역은 X).

지원 브라우저

  • Chrome (권장)
  • Microsoft Edge (권장)
  • Safari — 단일 HTML, ZIP 모드 OK. 폴더 모드는 webkitdirectory 지원 안 함.
  • Firefox — 폴더 모드 일부 제한.
